Description
===========

Allow generating avatar picture from ruby.



Examples
========

~~~ruby
require 'face-no-more'

FaceNoMore.generate(:cat, "foo@example.com", format: :jpg, size: 256)
FaceNoMore.generate(:bird, 12345678910)
~~~
